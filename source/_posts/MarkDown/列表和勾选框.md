# 列表

## 无序列表

（这里\*+-都可以，还是自动格式化问题）
（不可以混用）

- 第一项任务
- 第二项任务
- 第三项任务

## 有序列表

（必须连续，但是可以不从 1 开始）

1. 第一项任务
2. 第二项任务
3. 第三项任务

## 嵌套列表

（在列表中给一个 Tab）

1. 第一项任务
   - 嵌套一级无需列表
2. 第二项任务
   1. 嵌套一级有序列表
   2. 嵌套一级有序列表
3. 第三项任务

# 勾选框

([]内有 x 为打勾，空格为没打勾)

- [ ] 第一项任务
- [x] 第二项任务
- [ ] 第三项任务
