# C# Day1

## 1.装箱和拆箱是指什么?

**答案：**

- 装箱---把栈中内容迁移到堆中去(值转引用)
- 拆箱---把堆中内容迁移到栈中去(引用转值)

```cs
int i = 3;
object o = i;//装
i= (int) o;//拆
```
