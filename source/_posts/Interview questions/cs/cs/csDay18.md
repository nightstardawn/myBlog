# C# Day18

## 1.请问下面这段代码的打印结果是多少?

```cs
float f = 10.4f;
int i = f / 2;
Console.WriteLine(i);
```

**答案：**
该代码片段存在两种错误
应该改为

1. float f = 10.4f;
2. int i = (int)(f/2);
